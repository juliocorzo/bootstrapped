Bootstrap Buttons

Bootstrap buttons can be called using the custom Liquid tag btn. In order to give text, a link, and style to the button, each thing must be separated by  | , a | in between two spaces.

The correct syntax from creating a Bootstrap 4 button is: {% btn text | link | style %}. If no values are specified, the button will be rendered with default values, and the button style will be the primary color of the site.

{% btn %}
{% btn %}
{% btn Click Me %}
{% btn Click Me %}
{% btn Google | http://google.com %}
{% btn Google | http://google.com %}
{% btn Google | http://google.com | secondary %}
{% btn Google | http://google.com | secondary %}
{% btn Google | http://google.com | outline-dark %}
{% btn Google | http://google.com | outline-dark %}
